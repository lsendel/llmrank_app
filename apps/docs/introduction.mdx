# LLM Boost MCP Server

The LLM Boost MCP server gives your AI coding agent direct access to 27 SEO tools purpose-built for AI readiness. Crawl sites, score pages across 37 factors, track AI visibility, and get actionable fixes â€” all from within your editor.

## What is MCP?

The [Model Context Protocol](https://modelcontextprotocol.io) (MCP) is an open standard that lets AI assistants connect to external tools and data sources. LLM Boost provides an MCP server that exposes SEO analysis capabilities to any MCP-compatible client.

## Quick Start

Install the MCP server in under a minute:

<Tabs>
  <Tab title="Claude Code">
    ```bash
    claude mcp add llm-boost \
      --env LLM_BOOST_API_TOKEN=llmb_xxx \
      -- npx -y @llmrank.app/mcp
    ```
  </Tab>
  <Tab title="Cursor">
    Add to `~/.cursor/mcp.json`:
    ```json
    {
      "mcpServers": {
        "llm-boost": {
          "command": "npx",
          "args": ["-y", "@llmrank.app/mcp"],
          "env": { "LLM_BOOST_API_TOKEN": "llmb_xxx" }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Claude Desktop">
    Add to `claude_desktop_config.json`:
    ```json
    {
      "mcpServers": {
        "llm-boost": {
          "command": "npx",
          "args": ["-y", "@llmrank.app/mcp"],
          "env": { "LLM_BOOST_API_TOKEN": "llmb_xxx" }
        }
      }
    }
    ```
  </Tab>
</Tabs>

Then try: *"Audit the AI readiness of example.com"*
