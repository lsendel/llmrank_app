# Strategy Tools

## get_recommendations

Get a prioritized action plan to improve AI-readiness score. Recommendations are ranked by effort and impact.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Project ID |

**Example:**
```
"What should I fix first to improve my AI readiness score?"
```
