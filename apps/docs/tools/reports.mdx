# Report Tools

## generate_report

Generate a PDF or shareable report for a project crawl.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Project ID |
| `crawlId` | string (UUID) | Yes | Crawl job ID |

**Example:**
```
"Generate a report for my latest crawl that I can share with my team"
```
