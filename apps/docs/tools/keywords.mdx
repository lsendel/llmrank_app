# Keyword Tools

## discover_keywords

Discover AI-relevant keywords for your site. Identifies terms and phrases that AI assistants associate with your industry and content area.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Project ID |

**Example:**
```
"Discover keywords I should target for better AI visibility"
```
