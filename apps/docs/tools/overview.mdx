# Tools Overview

LLM Boost provides 27 tools organized by domain. Each tool is available to any MCP-compatible AI client.

## Project Management

| Tool | Description |
| --- | --- |
| `list_projects` | List all projects in your account |
| `get_project` | Get detailed project information |
| `create_project` | Create a new project for a domain |

## Crawling

| Tool | Description |
| --- | --- |
| `start_crawl` | Start a new site crawl |
| `get_crawl_status` | Check crawl progress |
| `list_crawls` | List crawl history for a project |

## Pages

| Tool | Description |
| --- | --- |
| `list_pages` | List all crawled pages |
| `get_page_details` | Get detailed page analysis |

## Scoring

| Tool | Description |
| --- | --- |
| `get_site_score` | Get overall site AI-readiness score |
| `compare_scores` | Compare scores between two crawls |
| `get_score_history` | View score trends over time |

## Issues

| Tool | Description |
| --- | --- |
| `list_issues` | List all issues found during a crawl |
| `get_fix_recommendation` | Get a fix recommendation for a specific issue |

## AI Visibility

| Tool | Description |
| --- | --- |
| `check_visibility` | Check how your site appears in AI responses |
| `list_visibility_history` | View visibility check history |

## Fixes

| Tool | Description |
| --- | --- |
| `generate_fix` | Generate an AI-powered fix for an issue |

## Strategy

| Tool | Description |
| --- | --- |
| `get_recommendations` | Get prioritized improvement recommendations |
| `get_content_gaps` | Identify content gaps vs competitors |

## Competitors

| Tool | Description |

## Reports

## Content
