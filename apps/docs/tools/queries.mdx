# Query Tools

## suggest_queries

Suggest visibility check queries based on your site's content and industry. Generates natural language queries that users might ask AI assistants.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Project ID |

**Example:**
```
"Suggest queries I should track for AI visibility"
```
