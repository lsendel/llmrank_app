# Competitor Tools

## list_competitors

List all tracked competitors for a project.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Project ID |

## compare_competitor

Compare your site's AI-readiness scores against a competitor.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `projectId` | string (UUID) | Yes | Your project ID |
| `competitorId` | string (UUID) | Yes | Competitor ID |

**Example:**
```
"Compare my site's AI readiness against competitor X"
```
